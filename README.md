# bdd.libertex

- [ELM Language](https://elm-lang.org)
- [Compiler](https://guide.elm-lang.org/install/elm.html)
- Test Runner: 
  - npm install -g elm-test
- Run tests:
  - elm-test
- Dev web server: 
    - elm reactor
    - browse to src/main.elm
- Additional dev tools:
  - VS Code with ELM extensions.
  - npm i -g elm-format
